Features: 
 - PC mode: 4 searches per 15 mins
 - Mobile Mode: 3 searches per 15 mins
 - Retreives data on Google Trends API

How to use:

SET:
Setting.XML
  - Username   :  your username
  - bpassword  :  your outlook password
  - region     :  google trends region
